# gh lgtmoon

GitHub CLI extension to retrieve LGTM images.

## Installation:

```sh
gh extension install YuG1224/gh-lgtmoon
```

The extension depends on jq.
```sh
brew install jq
```

## Usage:

```sh
gh lgtmoon
```
